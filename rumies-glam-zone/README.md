# Rumies Glam Zone 💄

A professional website for **Rumies Glam Zone** — a Nigerian makeup artist brand built with clean HTML, CSS, and JavaScript.

## 📁 Project Structure

```
rumies-glam-zone/
├── index.html
├── README.md
└── assets/
    ├── style.css
    ├── script.js
    └── images/
        ├── look-1.png
        ├── look-2.png
        ├── look-3.png
        ├── look-4.jpg
        └── look-5.jpg
```

## 🚀 Getting Started

1. Clone the repository
2. Open `index.html` in your browser — no build tools needed

## 🌐 Deploying

You can host this for free on:
- **GitHub Pages** — push to a repo, go to Settings → Pages → select `main` branch
- **Netlify** — drag and drop the folder on netlify.com
- **Vercel** — connect your GitHub repo

## ✏️ Customisation

| What to change | Where |
|---|---|
| Colours & fonts | `assets/style.css` → `:root` variables |
| Text & content | `index.html` |
| Photos | Replace files in `assets/images/` |
| Contact details | `index.html` → booking & footer sections |
